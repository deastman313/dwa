Hello World!

I'm Diana, nice to meet you.

During the daylight hours, I work in health economics research over at the Harvard School of Public Health. I have a strong interest in all things web, however, especially from the design perspective. I am taking this class to broaden my understanding of what is now capable on the web and to (hopefully) build some cool applications.

The layout of this page is built on Skeleton - a 960 grid system. I'm still trying to get up to speed with all of the new semantic HTML5 tags as well.

Project One.

The goal of this project was to get our workflows' nuts and bolts in place for future, more complex, coding projects.

Unfortunately, I purchased my web hosting plan approximately four years ago -- before I knew of Dreamhost's follies -- and inertia has prevented me from changing services.

To code up the HTML and CSS here, I used good old TextMate.
